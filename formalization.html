📖 Инструкция по использованию

Добро пожаловать! Этот инструмент помогает создавать красивые HTML-блоки.

📊 Режим: Таблица

  • Используйте кнопки сверху для управления структурой.
  • Магия иконок: Поставьте + в ячейку для галочки ✔ или - для крестика ✘ при копировании.
  • Горячие клавиши: Ctrl+Enter (строка), Ctrl+i (столбец), Ctrl+Del (уд. строку), Ctrl+Back (уд. столбец).

⚠️ Режим: Плашка

Яркий блок для важных предупреждений. Текст отображается в предпросмотре в реальном времени.

📦 Режим: Блоки

Сворачиваемые секции (аккордеоны). Используйте редактор для форматирования текста внутри.

📝 Режим: Список

Структурированный вывод: Заголовок, Описание и Маркированный список.
